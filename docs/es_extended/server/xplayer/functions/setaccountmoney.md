# xPlayer.setAccountMoney

```lua
xPlayer.setAccountMoney(account, money)
```

This function sets money for an account.

## 参数

| 参数 | 数据类型 | 数据类型 | 默认值 | 说明                                       |
|----------|-----------|----------|---------------|---------------------------------------------------|
| account  | string    | No       | -             | Valid accounts can be found in configuration file |
| money    | number    | No       | -             | Amount of money                                   |
