# xPlayer.addWeaponAmmo

```lua
xPlayer.addWeaponAmmo(weaponName, ammoCount)
```

This function adds the parsed ammo to the player weapon

## 参数

| 参数   | 数据类型 | 数据类型 | 默认值 | 说明 |
|------------|-----------|----------|---------------|-------------|
| weaponName | string    | No       | -             | Weapon name |
| ammoCount  | number    | No       | -             | Ammo to add |
