# xPlayer.triggerEvent

```lua
xPlayer.triggerEvent(eventName, ...)
```

该函数会触发玩家的客户端事件。

## 参数

| 参数      | 数据类型 | 数据类型 | 默认值 | 说明                    |
| --------- | -------- | -------- | ------ | ----------------------- |
| eventName | string   | No       | -      | Event name              |
| ...       | any      | Yes      | -      | Variable number of 参数 |
