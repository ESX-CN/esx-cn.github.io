# xPlayer.addWeapon

```lua
xPlayer.addWeapon(weaponName, ammo)
```

This function adds a weapon.

## 参数

| 参数   | 数据类型 | 数据类型 | 默认值 | 说明 |
|------------|-----------|----------|---------------|-------------|
| weaponName | string    | No       | -             | Weapon name |
| ammo       | number    | No       | -             | Ammo count  |
