# xPlayer.setJob

```lua
xPlayer.setJob(name, grade)
```

该函数设置了玩家的工作，这个工作必须在`jobs`数据库表中定义。

## 参数

| 参数  | 数据类型      | 数据类型 | 默认值 | 说明     |
|-------|---------------|----------|--------|----------|
| name  | string        | No       | -      | 工作名称 |
| grade | string&number | No       | -      | 工作等级 |
