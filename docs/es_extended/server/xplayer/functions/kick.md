# xPlayer.kick

```lua
xPlayer.kick(reason)
```

This function kicks a player with a reason.

## 参数

| 参数 | 数据类型 | 数据类型 | 默认值 | 说明                          |
|----------|-----------|----------|---------------|--------------------------------------|
| reason   | string    | Yes      | -             | Kick reason, will be shown to player |
