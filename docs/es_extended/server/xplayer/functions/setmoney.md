# xPlayer.setMoney

```lua
xPlayer.setMoney(money)
```

This function sets the player cash balance.

## 参数

| 参数 | 数据类型 | 数据类型 | 默认值 | 说明               |
|----------|-----------|----------|---------------|---------------------------|
| money    | number    | No       | -             | New money amount          |
