# ESX.Scaleform.ShowPopupWarning

```lua
ESX.Scaleform.ShowPopupWarning(title, msg, bottom, sec)
```

此函数显示基本的弹出警告，例如退出确认警告。

## 参数

| 参数   | 数据类型 | 可选项 | 默认值 | 说明                   |
| ------ | -------- | ------ | ------ | ---------------------- |
| title  | string   | No     | -      | 标题文字               |
| msg    | string   | No     | -      | 信息内容               |
| bottom | string   | No     | -      | 页脚文字               |
| sec    | number   | No     | -      | 显示时长（以秒为单位） |
