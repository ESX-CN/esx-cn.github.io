# ESX.Scaleform.ShowTrafficMovie

```lua
ESX.Scaleform.ShowTrafficMovie(sec)
```

This function starts the traffic scaleform movie used in the campaign.

## 参数

| 参数 | 数据类型 | 数据类型 | 默认值 | 说明                              |
| ---- | -------- | -------- | ------ | --------------------------------- |
| sec  | number   | No       | -      | Time in seconds to show scaleform |
