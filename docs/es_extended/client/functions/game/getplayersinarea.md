# ESX.Game.GetPlayersInArea

```lua
ESX.Game.GetPlayersInArea(coords, radius)
```

这个函数获取在一个给定的半径内的玩家。

## 参数

| 参数     | 数据类型   | 可选项    | 默认值        | 说明                         |
|----------|-----------|----------|---------------|------------------------------|
| coords   | vector3   | No       | -             | 要搜索的坐标                  |
| area     | number    | No       | -             | 最大搜寻距离                  |
