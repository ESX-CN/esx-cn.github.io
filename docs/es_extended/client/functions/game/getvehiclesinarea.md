# ESX.Game.GetVehiclesInArea

```lua
ESX.Game.GetVehiclesInArea(coords, radius)
```

这个函数返回一个给定的半径内的所有车辆。

## 参数

| 参数     | 数据类型   | 可选项    | 默认值        | 说明                          |
|----------|-----------|----------|---------------|------------------------------|
| coords   | vector3   | No       | -             | The coords to search at      |
| radius   | number    | No       | -             | The maxmimum search distance |
