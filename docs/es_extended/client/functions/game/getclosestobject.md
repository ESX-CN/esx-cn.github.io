# ESX.Game.GetClosestObject

```lua
ESX.Game.GetClosestObject(filter, coords)
```

此函数获取最近的对象。

## 参数

| 参数     | 数据类型   | 可选项    | 默认值        | 说明                                |
|----------|-----------|----------|---------------|-------------------------------------|
| filter   | table     | No       | -             | 具有对象名称的白名单表                |
| coords   | table     | No       | -             | 搜寻坐标                             |
