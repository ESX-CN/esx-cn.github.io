# ESX.Game.GetClosestVehicle

```lua
ESX.Game.GetClosestVehicle(coords)
```

此函数获取最近的载具。

## 参数

| 参数      | 数据类型   | 可选项   | 默认值            | 说明                      |
|----------|-----------|----------|-------------------|---------------------------|
| coords   | vector3   | Yes      | (玩家的坐标)       | 要搜索的坐标               |
