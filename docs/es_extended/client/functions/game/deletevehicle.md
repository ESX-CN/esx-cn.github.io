# ESX.Game.DeleteVehicle

```lua
ESX.Game.DeleteVehicle(vehicle)
```

该函数删除一辆已解析的载具。

## 参数

| 参数     | 数据类型   | 可选项    | 默认值        | 说明               |
|----------|-----------|----------|---------------|--------------------|
| vehicle  | string    | No       | -             | The vehicle handle |
