# ESX.Game.GetClosestPed

```lua
ESX.Game.GetClosestPed(coords, ignoreList)
```

此函数获取最近的ped。

## 参数

| 参数       | 数据类型   | 可选项    | 默认值        | 说明                                 |
|------------|-----------|----------|---------------|-------------------------------------|
| coords     | table     | No       | -             | 具有对象名称的白名单表                |
| ignoreList | table     | Yes      | {}            | An table of peds to ignore          |
